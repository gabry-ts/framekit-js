# Changelog

All notable changes to this project are documented in this file.

## 0.1.0

- Initial public release of FrameKit core APIs.
- Added compat entrypoint and migration-oriented verbs.
- Added DataFrame convenience methods (`assign`, `relocate`, `reify`, `lookup`, `spread`, `unroll`, `impute`).
- Added expression helpers including `op.corr` and `op.rowNumber`.
