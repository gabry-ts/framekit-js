# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |

## Reporting a Vulnerability

Report vulnerabilities privately by opening a private security advisory on GitHub or by contacting project maintainers directly.

Please include:

- Affected version
- Reproduction steps
- Impact assessment
- Suggested remediation (if known)

## Embargo and Disclosure Timeline

- Acknowledgement target: within 72 hours.
- Triage target: within 7 days.
- Fix target for high severity: within 30 days when feasible.
- Public disclosure follows coordinated release and patch availability.
